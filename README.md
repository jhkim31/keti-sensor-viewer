# Server (back-end)

## 1. requirements.txt 설치
server디렉토리 에서
```bash
pip3 install -r requirements.txt
```

## 2. 실행
```bash
python3 server.py
```


# Viewer (front-end)
## 1. npm install
viewer디렉토리 에서
```bash
npm install
```

## 2. config.js 수정

### /viewer/src/config.js
base_url 변경

## 2. npm 실행
```bash
npm start
```
